Enhance Moxo WordPress Themes functionality.

== Description ==

The Moxo plugin adds sections functionality to the Moxo theme and Others Moxo Market Themes.
This plugin for only Moxo Market. Moxo is a plugin build to enhance the functionality of WordPress Theme made by Moxo Market.

The Moxo plugin features include:

- Premade Section with Dummy Data
- One Click Data Installation
- 4 predefined content sections
- Live content editing 
- Custom Theme Color
- and many other features
